1. Install iterm2 https://www.iterm2.com/

2. Install homebrew https://brew.sh/
3. Install oh-my-zsh https://ohmyz.sh/
4. Install scm_breeze
     
    a. create ~/.bashrc 
       
       $ touch ~/.bashrc
